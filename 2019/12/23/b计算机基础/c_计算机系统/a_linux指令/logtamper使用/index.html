这里对-m参数补充说明一下：

-m 操作的模式 固定值，3个可选【1/2/3】
分别是

  • 1：修改当前登陆用户。
    • 日志文件：_var_log/wtmp
    • 查看命令： who
  • 2：清除登陆日志。
    • 日志文件：_var_run/utmp
    • 查看命令：last | more
  • 3、不是清除，是修改覆盖上次的登陆信息，要修改的用户为-u后面的参数，改为后面 -i -t -d的信息。所以，例如，想改root用户，则-u root，-i -t -d随便输入。
    • 日志文件：_var_log/lastlog
    • 查看命令：lastlog
历史命令：history
