switch注意事项

1. dk1.6中switch不可以字符串 ，double 都不支持
2. 条件-确定区间的开头，break决定退出
3. 不要少break
4. 于if的不同：
    1. switch 可以转换成if ，但是if不一定能转成switch
    2. switch 有类型的限制，而if没有

for 和 switch连用，break和continue的特点

  1. break不是跳出for而是跳出switch
  2. continue和if中的一样，后面的不执行了
