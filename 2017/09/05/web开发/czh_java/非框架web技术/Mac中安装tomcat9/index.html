在iterm中对路径下的tomcat进行权限配置


在给startup.sh配置权限的时候，提示也要对catalina.sh文件权限配置


我在第二次启动tomcat的时候，会提示logs/catalina.out：permissin denied
这时切到logs目录下，chmod 777 catalina.out就行了
