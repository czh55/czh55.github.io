JS遇到问题怎么办

  1. 对出问题的地方去查手册，看手册中没有说限制
  2. 很多时候都是浏览器的问题
    1. 不同浏览器的效果不同
    2. 方法执行结果不同
    3. 有些方法在部分浏览器中不能用
  3. 要看开发者工中的报错
  4. js中的有些事件是不按代码顺序走的，而是按触发来执行的，这一点区别于其他语言，一定要小心
